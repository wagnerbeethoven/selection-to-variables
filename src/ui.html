Selection to Variables

Read the current selection, review raw values, and turn them into Figma variables.

UI v3
How this plugin works Step-by-step guidance
1. Select Groups, Frames, or Sections The plugin only scans those top-level selection types.
2. Read selection It extracts colors, text values, sizes, spacing, and repeated values from the selected content.
3. Review all tokens Tokens are shown in Colors, Texts, and Sizes groups, and each group opens with all scanned rows visible.
4. Keep naming compact Suggested names are shorter now, but you can still edit them before export or creation.
5. Export or create variables After reviewing the scanned tokens, export JSON or create Figma variables and bind them back to the selection.
